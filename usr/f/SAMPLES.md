# Optional audio sample packs

Sample libraries under `samples/` are **gitignored** and installed locally by the user. They are not part of the Apache-2.0 source release.

## BSR Samples Yellow (BSRSamples001)

Install from the vendor zip into:

```
samples/bsr_yellow/BSR_Samples_Yellow_BSRSamples001/
```

Use with `synth.load_sample()` — see [SYNTH.md](SYNTH.md) and [examples/synth_bsr_sample.ie](../examples/synth_bsr_sample.ie).

Verify the sample-pack license with the vendor before redistribution or release.
